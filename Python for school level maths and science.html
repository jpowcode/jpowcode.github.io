Python for A level Mathematicians and Scientists¶

This document will take you through some programming concepts that you might find useful whilst studying A-level maths and science subjects. It is not difficult to get started programming, but it is an incredibly useful skill.

It will cover lots of topics such as performing algebraic calculations, calculating with units and error bounds, analysing data and plotting graphs.

The document is made up of individual cells. Some contain text like this one. Some contain code like the next one that can be run by clicking the play icon on the left of the cell. Go ahead and run the next cell. It just does some routine stuff that we will need. You don't need to worry about what it contains. Just move straight past onto the next cell titled "Expanding Brackets".

Expanding brackets¶

The expand function can be used to help you check your algebra or to perform complicated expansions. Make sure you put a * in between any brackets to multiply them. For example you can expand the following
$(x+1)(x+2)(x+3)(x+4)$
$(x+3)(x+5)^3$
$(x+3)^3$

In [0]:
out[1] = s.expand((x+1)*(x+2)*(x+3)*(x+4))
out[2] = s.expand((x+3)*(x+5)**2)
out[3] = s.expand((x+3)**3)
pprint(1,2,3)
Out[0]:
$$x^{4} + 10 x^{3} + 35 x^{2} + 50 x + 24\\x^{3} + 13 x^{2} + 55 x + 75\\x^{3} + 9 x^{2} + 27 x + 27\\$$

Simplifying expressions¶

The simplify function can be used to simplify algebraic expressions. For example you can simplify
$x^4+x^3+x^2-3x^4+4x^3$

In [0]:
out[4] = s.simplify(x**4 +x**3 + x**2 -3*x**4 +4*x**3)
pprint(4)
Out[0]:
$$x^{2} \left(- 2 x^{2} + 5 x + 1\right)\\$$

Variables¶

You may have wondered what the out[] is for. out is a variable. It is a particular type of variable called a list. We ar using a programming language called Python. Click on the links for more information on variables and lists. In Python you can make any expression or number a variable and make use of it later in your programme. For example

In [0]:
a = x**2 + y**2
b = x + y
out[5] = s.expand(a*b)

Out[0]:
$$x^{3} + x^{2} y + x y^{2} + y^{3}\\$$

Solving equations¶

We can use python to solve equations. For example we can solver the equation $x^2 - 1 =0$. To do this we use the solve function. In the example below. Notice how the =0 is missed out. This is because the solve function can only solve equations that are equal to zero. Notice also that we need to tell the solve function what the variable in the equation is by putting in a comma followed by the x.

In [0]:
out[6] = s.solve(x**2 - 1, x)
pprint(6)
Out[0]:
$$\left [ -1, \quad 1\right ]\\$$

Solving simultaneous equations¶

This is very similar to solving one variable equations. All equations must be of the form $=0$ and we must tell the solve function what the different variables are. For example, we can solve
$x+5y=2$
$-3x+6y=15$
First we have to rearrange them to make them $=0$
$x+5y-2=0$
$-3x+67-15=0$

In [0]:
out[14] = s.solve((x + 5*y - 2, -3*x + 6*y - 15), x, y)
pprint(14)
Out[0]:
$$\left \{ x : -3, \quad y : 1\right \}\\$$

out[15] = m.sin(m.pi/2)

Differentiation¶

You can find derivatives. For example
$\frac{d}{dx}\cos^2(x)$
$\frac{d^3}{dx^3}x^4$

In [0]:
out[21] = s.diff((s.cos(x))**2, x)
out[22] = s.diff(x**4, x, 3)
pprint(21,22)
Out[0]:
$$- 2 \sin{\left (x \right )} \cos{\left (x \right )}\\24 x\\$$

Integration¶

You can also integrate. Note how we make the infinity symbol oo. For example
$\int \cos(x) dx$
$\int_0^{\infty}e^x dx$

In [0]:
out[23] = s.integrate(s.cos(x), x)
out[24] = s.integrate(s.exp(-x), (x, 0, s.oo))
pprint(23,24)
Out[0]:
$$\sin{\left (x \right )}\\1\\$$

Vectors and matrices¶

You can define matrices and vectors. Multiply them, find inverses and determinants. For example if
A = $\begin{pmatrix} 1 & -1 \\ 3 & 4 \\ 0 & 2 \end{pmatrix} B = \begin{pmatrix} 1 & 2 & 4 \\ 3 & 4 & -2 \end{pmatrix} C = \begin{pmatrix} 1 & -1 \\ 3 & 4 \end{pmatrix}$
Find $A \cdot B, \det{B},$ and $C^{-1}$

In [0]:
A = s.Matrix([[1, -1], [3, 4], [0, 2]])
B = s.Matrix([[1, 2, 4], [3, 4, -2]])
C = s.Matrix([[1, -1], [3, 4]])

out[26] = C.det()
out[27] = C**-1

Units¶

We can make calcualtions using different units by using a library called Pint. For example, lets say we want to calculate a speed from a distance in kilometers and a time in minutes, but we want the answer in meters per second. There are hundreds of units including anything you are likely to ever want in science. A full list an be found here.

In [0]:
d = 24.0 * units.kilometers
t = 3 * units.minutes
speed = d/t
speed.ito(units.meters / units.seconds)

print(speed)

133.33333333333334 meter / second

Prefixes¶

We can deal with prefixes simply by adding in to the unit defintinition. All of the standard scientific units are available. For example, the above example modified with micro-meters and yocto-seconds with a conversion to meters per second.

In [0]:
d = 3.5 * units.micrometers
t = 2.1 * units.yoctoseconds
speed = d/t
speed.ito(units.meters / units.seconds)
print(speed)

1.6666666666666668e+18 meter / second

Error Bounds¶

Lets say we have the equation for gravitation force between two masses
$F_g = \frac{Gm_1m_2}{r}$
and we and we have the values for $G,m_1, m_2$ and $r$ defined as follows with error bounds given
$m_1 = 3.5 \pm 0.1$ mega kilogrammes
$m_1 = 2.1 \pm 0.2$ mega kilogrammes
$r = 73000 \pm 1000$
Lets say we also want to convert the answer into base units.

In [0]:
m1 = ufloat(3.5, 0.1) * units.megakilogram
m2 = ufloat(2.1, 0.2) * units.megakilogram
r = ufloat(73000,1000) * units.kilometer
G= sc.constants.G
F = G*m1*m2/r
print(F.to_base_units())

(6.7+/-0.7)e-06 kilogram ** 2 / meter

This example uses a function called uloat which comes from a library called uncertainties.
